01 Device Layer — User's Mac
v0.1
🖥
Tauri Shell
Native macOS app. Process lifecycle, auto-updates, system tray. Minimal IPC allowlist — webview cannot access filesystem, shell, or network directly.
v0.1
📦
Custom PDS Shell
Purpose-built embedded PDS. SQLite-backed, local-first. Owns XRPC surface and sync protocol. The core technical risk — must federate on the live network before anything else matters.
v0.1
🧩
Dependency Stack
atrium-api — XRPC types, lexicon defs (auto-generated). atrium-repo — MST read/write, CAR export. rsky-crypto — P-256/K-256 commit signing.
v0.1
🔗
Iroh Endpoint
QUIC-based tunnel to relay. NAT traversal, connection resumption on wake. Pushes signed repo commits to relay when online.
v0.1
🔐
DID Keystore
Signing keys in macOS Keychain. Basic key management for v0.1. Shamir split added in v1.0.
v1.0
🛟
Recovery Share Manager
Generic UI for managing Shamir shares. Supports iCloud Keychain, encrypted file export, relay escrow. Interface designed for future social recovery.
v1.0
⚠️
Compat Warning Banner
Non-blocking in-app warning when spec drift detected. Links to update. Never blocks launch.
v1.0
🛡
XRPC Hardening
Request size limits on all endpoints. Rate limiting at relay. cargo-fuzz targets for CBOR/CAR/MST parsing paths. Adversarial MST key distribution testing per ATProto spec guidance.
REMOVED
📦
rsky-pds (fork)
Replaced by custom PDS shell in v5. Now tracking a spec, not a codebase.
02 Relay Layer — Managed + BYO
Managed Relay (Your Infrastructure)
v0.1
🚇
Iroh Relay Node
Single-region for v0.1. Always-on tunnel endpoint. Receives commits from device, proxies XRPC repo reads.
v0.1
📣
requestCrawl Trigger
On device reconnect, pings BGS requestCrawl so new content propagates immediately.
v1.0
📡
Firehose Proxy
Persistent WebSocket to BGS on behalf of PDS. Streams commits continuously. BGS sees 100% uptime. Key paid feature. Beta/preview at launch.
v1.0
💾
Commit Buffer
Rolling log of signed repo commits. Feeds firehose proxy during offline. Tiered retention: 7d free, 30d paid, 90d business.
v1.0
⚙️
Provisioning API
Account setup, domain linking, relay config. Onboarding flow for new Tauri app connections.
v1.0
🔏
Key Share Escrow
Holds one encrypted Shamir share. Cannot reconstruct alone. Encrypted at rest, access-logged.
v1.0
💓
Health Monitor
Device liveness, relay uptime, ATProto spec compat. Includes canary account for silent federation failure detection.
LATER
🌎
GeoDNS Multi-Region
2-3 relay nodes, route to nearest healthy. Simple failover with brief firehose gap. Cross-region replication interface designed but not built.
LATER
🗄
Repo Snapshot
Full repo backup on relay. Incremental from commit buffer. Pro+ feature. Enables one-click device migration.
LATER
🌐
CDN / Public Cache
Serves public repo content during offline windows.
BYO Relay (User-Hosted, Free)
v1.0
📦
Relay Binary
Open-source. Nix flake (source of truth) → Docker image + NixOS module. Tunnel + commit forwarding + requestCrawl. No firehose proxy, no snapshots.
v1.0
📜
Device-Relay Protocol Spec
Documented contract: handshake/auth, commit push, health ping, optional feature negotiation. Includes commit ack for future trust verification.
v1.0
🔌
Feature Negotiation
App queries relay capabilities on connect. Gracefully degrades when extended features unavailable. Suggests upgrade for missing features.
REMOVED
📬
Inbound Message Queue
Not needed. ATProto records live in author's repo.

Runtime Threat Mitigations
v1.0
🔒
XRPC Input Hardening
Request size limits per endpoint. Rate limiting at relay layer. cargo-fuzz targets for CBOR/CAR/MST parsing. Adversarial MST key testing per spec DoS guidance.
v0.1
🏗
Tauri IPC Lockdown
Minimal allowlist: create/list/get records + status. Webview cannot access filesystem, shell, HTTP, or crypto. All sensitive ops in Rust backend only.
LATER
🤝
Relay Trust Verification
Device verifies commits appear in firehose. Protocol designed now (commit ack with seq number), verification logic built later. Protects against censorship by relay.
