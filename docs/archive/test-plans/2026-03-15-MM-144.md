## End-to-End: Full Happy Path

Complete the full user journey in one pass:

1. Start relay: `cargo run -p relay` from workspace root.
2. Seed a fresh claim code via Bruno collection (`POST /v1/admin/claim-codes`). Note the code.
3. Launch app: `cargo tauri ios dev`.
4. Welcome screen → tap "Get Started".
5. Claim Code → enter seeded code, verify button enables at 6 chars → tap "Next".
6. Email → enter "fulltest@example.com", verify button enables on valid format → tap "Next".
7. Handle → enter "fulltestuser", verify button enables → tap "Create Account".
8. Loading → verify spinner + "Creating your account…" text.
9. Success → verify "Account Created!" / "DID ceremony coming soon…".
10. Relay logs → confirm 201 response, correct 5-field request body.
11. Keychain → verify `device-private-key` (32 bytes), `device-token`, `session-token` under service `"ezpds-identity-wallet"`.

## End-to-End: Error Recovery Flow

Validate that the user can recover after an error without restarting the app:

1. Start relay with no claim codes seeded.
2. Complete onboarding with code "BADBAD", email "recover@example.com", handle "recoveruser".
3. Verify app rewinds to Claim Code screen with the expired error message.
4. Seed a valid claim code "GOOD01" in the relay.
5. Clear claim code input, enter "GOOD01". Tap "Next".
6. Email screen should still show "recover@example.com" (form state preserved). Tap "Next".
7. Handle screen should still show "recoveruser". Tap "Create Account".
8. Verify Loading screen appears and flow completes with "Account Created!".
