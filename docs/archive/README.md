# Archive

Historical documents kept for the record — nothing in here describes the
current system authoritatively.

- **`design-plans/`, `test-plans/`, `implementation-plans/`** — dated plans
  whose work has fully landed (or was superseded). They are immutable
  historical records: when a plan's work ships and it stops being referenced
  by active work, move its whole design/test/implementation triad here
  together so their relative cross-links keep working. Plans still in flight
  stay in `docs/{design,test,implementation}-plans/`.
- **Superseded spec versions** (`.docx`/`.html` exports and `*-amendments.md`
  notes) — earlier revisions of the living specs at `docs/` root.
- **`cross-spec-analysis.md`** — analyzed the v7/v0.2/v1.2 spec generation
  archived alongside it; resolved by `docs/unified-milestone-map.md`.

Architectural decisions extracted from these documents live in the ADR log:
[`../architecture/decisions/`](../architecture/decisions/README.md).
