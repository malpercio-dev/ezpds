# Keep ATProto extension in sync with relay capabilities
When adding new endpoints or capabilities to the ezpds relay, update the ATProto pi extension at `.pi/extensions/atproto/index.ts` to expose them as tools. The extension serves as the programmatic interface for testing and automation. Convention agreed upon 2026-06-22.
